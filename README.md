# Sync Family Bot 🛒

Telegram бот для синхронизации списка покупок в семье. Позволяет создавать семейные группы, добавлять продукты, отмечать
покупки и редактировать список в реальном времени.

## ✨ Возможности

- 👨‍👩‍👧‍👦 **Создание семейной группы** - создайте свою семью и приглашайте участников
- 🔗 **Приглашение по ссылке** - каждый участник может присоединиться по уникальной ссылке
- 📝 **Добавление продуктов** - отправьте сообщение с продуктами через запятую или с новой строки
- ✅ **Отметка покупок** - нажимайте на кнопки, чтобы отметить купленные товары
- ✏️ **Редактирование списка** - режим редактирования позволяет изменять названия и удалять продукты
- 🗑 **Очистка списка** - одним нажатием очистить весь список покупок
- 🔔 **Мгновенные уведомления** - все участники видят обновления в реальном времени

## 🚀 Технологии

- **Go 1.26** - основной язык разработки
- **Telebot v3** - библиотека для Telegram Bot API
- **PostgreSQL 15** - база данных
- **Jet ORM** - генерация type-safe SQL запросов
- **SQLx** - расширение для database/sql
- **Docker** - контейнеризация приложения
- **GitHub Actions** - CI/CD пайплайн

## 🛠 Установка и запуск

### Локальная разработка

1. **Клонируйте репозиторий**

```bash
git clone https://github.com/yourusername/sync_family_bot_go.git
cd sync_family_bot_go
```

2. **Создайте файл окружения**

```dotenv
BOT_TOKEN=your_telegram_bot_token
DATABASE_URL=postgres://bot:bot@localhost:5432/bots
 ```

3. **Запустите PostgreSQL**

```bash
   docker run -d \
   --name postgres \
   -e POSTGRES_DB=bots \
   -e POSTGRES_USER=bot \
   -e POSTGRES_PASSWORD=bot \
   -p 5432:5432 \
   postgres:15
```

4. **Установите зависимости**

```bash
go mod download
```

5. **Установите утилиты**

```bash
go install github.com/pressly/goose/v3/cmd/goose@latest
go install github.com/go-jet/jet/v2/cmd/jet@latest
```

6. **Запустите миграции**

```bash
make migrate-up
# или вручную:
goose -dir migrations postgres "postgres://bot:bot@localhost:5432/bots?sslmode=disable" up
```

7. **Сгенерируйте Jet модели**

```bash
make gen-db
# или вручную:
jet -source=PostgreSQL -dsn="postgres://bot:bot@localhost:5432/bots?sslmode=disable" -schema=family_sync -path=./internal/gen
```

8. **Запустите бота**

```bash
go run main.go
```

**Docker Compose**

```yaml
version: '3.8'
services:
  postgres:
    image: postgres:15
    environment:
      POSTGRES_DB: bots
      POSTGRES_USER: bot
      POSTGRES_PASSWORD: bot
    ports:
      - "5432:5432"
    volumes:
      - postgres_data:/var/lib/postgresql/data

  bot:
    build: .
    depends_on:
      - postgres
    environment:
      BOT_TOKEN: ${BOT_TOKEN}
      DATABASE_URL: postgres://bot:bot@postgres:5432/bots?sslmode=disable
    env_file:
      - .env

volumes:
  postgres_data:

```

# 📱 Использование Sync Family Bot

## 🚀 Быстрый старт

После запуска бота отправьте `/start` - бот проверит, состоите ли вы в семье, и предложит соответствующие действия.

## 📋 Команды

| Команда          | Описание                              | Пример           |
|------------------|---------------------------------------|------------------|
| `/start`         | Начать работу с ботом                 | `/start`         |
| `/create_family` | Создать новую семью                   | `/create_family` |
| `/start <код>`   | Присоединиться к семье по приглашению | `/start abc123`  |

## 👨‍👩‍👧‍👦 Управление семьей

### Создание семьи

1. Отправьте команду `/create_family`
2. Бот создаст новую семью и сгенерирует уникальную ссылку-приглашение
3. Нажмите кнопку "👪 Отправить приглашение", чтобы поделиться ссылкой с членами семьи

### Присоединение к семье

1. Получите ссылку-приглашение от члена семьи
2. Перейдите по ссылке или скопируйте код после `/start`
3. Бот автоматически добавит вас в семью

### Выход из семьи

1. Нажмите кнопку "🚪 Выйти из группы" в главном меню
2. Подтвердите действие (функция в разработке)

## 🛒 Управление списком покупок

### Добавление продуктов

**Способы добавления:**

1. **Через запятую** - отправьте сообщение с продуктами, разделенными запятыми - Молоко, хлеб, яйца, масло, сыр
2. **Многострочный список** - каждый продукт с новой строки
3. **Смешанный формат** - комбинация запятых и новых строк
4. **Одиночный продукт** - просто отправьте название

**Особенности:**

- Первая буква продукта автоматически становится заглавной
- Повторяющиеся продукты не дублируются
- Пустые строки и лишние пробелы игнорируются
- Все участники видят обновленный список мгновенно

**Результат:**

### ✅ Отметка покупок

* **Как отметить:** В обычном режиме просто нажмите кнопку с названием продукта.
* **Статусы:**
    * Не куплен: `Молоко`
    * Куплен: `✅ Молоко` (повторное нажатие снимет отметку).
* **Уведомления:**
    * При покупке: *🛒 Анна купил(а): Молоко*
    * При отмене: *🛒 Анна отменил(а) покупку: Молоко*

### 📊 Порядок сортировки

1. Сначала отображаются **некупленные** продукты.
2. Затем — **купленные** (отмеченные ✅).
3. Внутри каждой группы сортировка идет **по дате добавления**.

---

### ⚙️ Режим редактирования

* **Вход:** Нажмите кнопку `⚙️ Редактировать список`.
* **Выход:** Нажмите `⬅️ Назад к покупкам`.

#### ✏️ Изменение названия

1. В режиме редактирования нажмите на нужный продукт `✏️ Название`.
2. В меню выберите `📝 Изменить`.
3. Введите новое название.

#### 🗑 Удаление продукта

1. В режиме редактирования нажмите на продукт.
2. Выберите `🗑 Удалить`.
    * *Уведомление: 🗑 Анна удалил(а) товар: Молоко*

---

### 🏁 Очистка всего списка

1. Нажмите кнопку `🏁 Завершить покупки`.
2. Подтвердите действие: `✅ ДА, УДАЛИТЬ`.
    * *Уведомление: 🗑 Анна очистил(а) список покупок*

---

### 🔔 Синхронизация

* **Мгновенные обновления:** Все изменения сразу видны всем участникам.
* **Автоматизация:** Старые сообщения автоматически удаляются.
* **Типы уведомлений:** Бот сообщает о добавлении, покупке, удалении и очистке.

---

### 💡 Советы

* **Массовое добавление:** Используйте запятые для добавления списка продуктов.
* **Гибкость:** Редактируйте названия вместо удаления и создания заново.
* **Чистота:** Используйте «Завершить покупки» для очистки списка.

---

### ❓ Частые вопросы

* **Как пригласить участников?** Нажмите `🔗 Получить ссылку приглашение`.
* **Видят ли другие мои действия?** Да, все синхронизируется мгновенно.
* **Можно ли восстановить удаленное?** Нет, удаление необратимо.
